# @signalops/flow-feature-compare

> The `/compare` screen. Part of the **Flow** variant (`packages/flow/*`).

Signal selector + re-run, the before/after attribute diff, the change timeline and the
user-impact card.

## Commands

```bash
pnpm nx run flow-feature-compare:typecheck
pnpm nx run flow-feature-compare:test
pnpm nx run flow-feature-compare:lint
pnpm nx run flow-feature-compare:build
```

See [`docs/flow/architecture.md`](../../../docs/flow/architecture.md) for the
package's place in the dependency graph and the boundaries enforced around it.
