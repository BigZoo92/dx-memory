# @signalops/flow-feature-incidents

> The `/incidents` screen. Part of the **Flow** variant (`packages/flow/*`).

Summary KPI cards, status/severity/impact filters (+ reset), and the incidents table with
linked-signal navigation.

## Commands

```bash
pnpm nx run flow-feature-incidents:typecheck
pnpm nx run flow-feature-incidents:test
pnpm nx run flow-feature-incidents:lint
pnpm nx run flow-feature-incidents:build
```

See [`docs/audit/flow/flow-boundaries.md`](../../../docs/audit/flow/flow-boundaries.md) for the
package's place in the dependency graph and the boundaries enforced around it.
